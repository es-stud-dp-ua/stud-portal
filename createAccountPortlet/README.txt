Installation steps:
 1. Package createAccountHook;
 2. Deploy createAccountHook on your portal;
 3. Package createAccountPortlet;
 4. Deploy createAccountPortlet on your portal;
 5. Add new private page with url address <your site>/registration;
 6. Add createAccountPortlet in private page /registration.
 7. Enjoy!

Now if you following "Create Account" link on login portlet can see your createAccountPortlet.
